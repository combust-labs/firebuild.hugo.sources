# kill

Gracefully terminates a running virtual machine.

## Flags

- `--profile`: name of the profile to use, no default
- `--vmm-id`: ID of the virtual machine to inspect

## Example

```sh
sudo firebuild kill \
    --profile=standard \
    --vmm-id=wcabty1922gloailwrce
```
